## 🎨 Visual Design & UI System

* **Cyberpunk + Clean Premium Theme Blend**: Sleek dark futuristic interface that transitions smoothly into a modern clean-white design.
* **Light / Dark Theme Toggle**: Interactive navigation bar switch that stores user preference locally in browser memory (`localStorage`).
* **Holographic Portrait Avatar**: Circular profile picture equipped with visual cursor focus animations, filter increases on hover, and absolute game-character style HUD overlays.
* **Responsive Layouts**: Designed mobile-first utilizing fluid CSS grid configurations.
* **Parallax 3D Card Tilts**: Mouse hover transformations that dynamically calculate depth coordinates on project, education, and skill panels.
* **Background Particle Canvas**: High-performance interactive background connecting canvas nodes that respond to cursor movement.

## 💻 Running the Project Locally

If you wish to run this website on your local machine:
1. Clone this repository:
   ```bash
   git clone https://github.com/Revu-15/Protofilo.git
   cd Protofilo
   ```
2. Start a local server (Python 3 is recommended):
   ```bash
   python -m http.server 8000
   ```
3. Open your browser and navigate to: **`http://localhost:8000`**
